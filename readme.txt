== Description ==

One Page Quick Checkout for WooCommerce adds popup checkout, direct checkout buttons, one-page checkout templates, checkout field controls, and a floating cart drawer for WooCommerce stores.

= Key Features =

* Popup checkout form for completing purchases without leaving the current page.
* Checkout form field, label, placeholder, and required-field controls.
* Product quantity controls and product removal inside the checkout form.
* Menu cart with a slide-out drawer.
* Single-product one-page checkout.
* Multiple product checkout templates including table, list, single, slider, accordion, tabs, and pricing table layouts.
* Quick checkout buttons for product and archive pages.
* AJAX-powered cart and checkout interactions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/one-page-quick-checkout-for-woocommerce/` directory, or install the plugin through the WordPress Plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Make sure WooCommerce is installed and active.
4. Configure the plugin from the Onpage Checkout admin menu.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes. WooCommerce must be installed and active.

= Can I use this for specific products only? =

Yes. You can enable one-page checkout on individual product edit screens or use the plugin shortcodes and blocks.

= Does it support variable products? =

Yes. The plugin includes support for variable products and variation selection.
